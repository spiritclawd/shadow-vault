How It Works

Three instructions. Total privacy.

Shadow Vault is a Solana program with three core operations. No admin keys. No upgrade authority after deployment. Immutable.

01

Initialize

Create a privacy pool with a unique seed. Generates a PDA vault that only the owner can control.

02

Deposit

Lock SOL with a cryptographic commitment. The on-chain record shows a hash — not an amount.

03

Execute

Agent trades using encrypted balance. Policy engine enforces limits without revealing positions.

04

Withdraw

Redeem with a nullifier proof. Prevents double-spend without linking deposits to withdrawals.
